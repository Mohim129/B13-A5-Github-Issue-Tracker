3️⃣ What is the difference between map(), filter(), and forEach()?
Ans: Here are the difference between map(), filter(), and forEach():-
    |map(): It Creates a new array by applying a function to each element.
    |filter(): It creates a new array with elements that pass a condition
    |forEach(): It runs a function on each element but doesn't return a new array.
